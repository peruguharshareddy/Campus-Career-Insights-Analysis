# Campus Career Insights Analysis

This project analyzes student placement data using **Python (Pandas)** and **Power BI** to generate insights about placement trends, salary distribution, and internship impact.

## Project Overview

The goal of this project is to clean and analyze placement data and visualize key insights using Power BI dashboards.

## Tools & Technologies

* Python
* Pandas
* Power BI
* CSV Dataset

## Project Workflow

1. Raw dataset (`placement_data.csv`)
2. Data cleaning using Python (`data_cleaning.py`)
3. Clean dataset generated (`cleaned_data.csv`)
4. Data visualization using Power BI (`career_analysis.pbix`)

## Dashboard Preview

The final Power BI dashboard visualizes:

* Placement distribution
* Salary trends by department
* Internship impact on placements

See **Dashboard.png** for the dashboard screenshot.

## How to Run the Project

1. Install Python and Pandas
2. Run the cleaning script:

```
python data_cleaning.py
```

3. Open the Power BI file:

```
career_analysis.pbix
```

This will display the analytics dashboard.
